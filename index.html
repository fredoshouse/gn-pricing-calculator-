How to use this thing

Quick reference. Anything not covered here, just ask Fredo.

The Flow

Set the Total Budget, Start Month, and Duration. Pick a team preset or add roles one by one. Type hours per person per month. The schedule does the math.

Role Dropdown & Rates

Each row has a role picker grouped into C-Suite, Strategy, Creative, Ops/Account, Freelance/Partner. Picking a role auto-applies the rate from the 2025 GN rate card. Rates are still editable for one-offs.

The PT Toggle

The yellow PT button on each labor row flags that person as passthrough — freelance or external. Their cost flows into the Passthrough total instead of Labor. Use this for Jahred, Tonja, Corinne, Jarrod, or anyone billed through GN but not on staff.

Passthrough Block

Hard costs that aren't people — production, paid media, talent, seeding, licensing. Pick a category from the dropdown or type your own. Add a Note for context ("Apr–May only", "PS92").

Margin Grade

A+ Excellent (≥50%), A Healthy (40–49%), B OK (30–39%), C Compressed (20–29%), D Risk (10–19%), F Loss (< 0% or below target). Color and badge update live. Reminders panel calls out specifics.

Phase Budget

Split the total budget across phases (Discovery 20%, Strategy 25%, etc.). Useful when prepping the SOW so you can defend the allocation.

Payment Schedule

Milestones with percentages, due months, status (Pending/Invoiced/Paid/Overdue), and notes. Presets: 30/40/30, 50/50, 3 × equal, Equal monthly, Quarterly — or build it custom. Amounts auto-compute from the total budget. Status colors persist into the Excel export so it can double as a tracker.

Deal Status & Project Type

Tag every deal as Draft / Sent / Negotiating / Signed / Lost with a colored badge. Project Type (Retainer, Campaign, Brand Strategy, Origination, etc.) sets you up for cross-deal search and "show me similar deals" in Phase 2.

Internal Notes

Strategic context, why you priced this way, who's championing it, what's negotiable. Private to you. Loaded back automatically when you re-open the deal.

Minimize Any Panel

The − button in every panel collapses it. Collapsed state is saved per deal — so a Signed deal can ship with everything collapsed except Summary if that's how you like it.

Duplicate a Deal

Loaded a deal? The Duplicate button clones it with a new name and resets status to Draft. Use it as a starting template for a similar deal — saves rebuilding the schedule from scratch.

Saving & Loading

Auto-saves to this browser on every change and when you click out of the tab. Green dot = saved, yellow = saving. Hit Save now to force a save. Type a name in Save as… and hit Save to keep a named copy. Export JSON for cross-machine moves.

Excel Export

Generates a real .xlsx with live formulas — open in Excel or Google Sheets, change a cell, totals recalc. Includes labor, passthrough, summary, fit, phase, payment schedule, and notes. Drop into Drive and Drive auto-converts to Sheets.

Per-Deal Lock

Click 🔒 Set password while a deal is loaded to lock it. Anyone trying to load it later is prompted. Note: light obfuscation, not real encryption — stops casual viewing, not someone with browser dev tools.

Password Visibility

The 👁 button on the site password gate (and any prompt) toggles between hidden dots and visible text — useful when you mistype.

Rates & Steps

Type any rate. Arrow keys / spinners move in increments of $5 for fine control. Switch a role and the rate auto-applies from the rate card; override if needed.

Mobile / Tablet

Layout adapts. The schedule scrolls horizontally — swipe to see all months. Best experience is still on desktop.
